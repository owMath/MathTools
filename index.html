Sobre os Algoritmos

Sum8 / Sum16 / Sum32

Soma simples de todos os bytes, com resultado truncado para 8, 16 ou 32 bits. Usado em muitas ECUs Bosch e Delphi.

XOR8

Operação XOR de todos os bytes. Comum em protocolos de comunicação serial com ECU (K-Line, ISO 9141).

CRC-16/CCITT

Polinômio 0x1021 com valor inicial 0xFFFF. Usado em protocolos CAN e em verificação de firmware de ECUs modernas.

CRC-32

Polinômio padrão IEEE 802.3. Usado em verificações de integridade de arquivos de calibração e firmware completos.

Regiões Típicas do ME7

ROMSYS (Boot)

Checksum da área de inicialização. Geralmente nos primeiros 0x4000 bytes. Algoritmo Sum16.

Main Program

CRC32 do programa principal. Cobre a maior parte do firmware. Armazena o CRC no header.

Multipoint Blocks

Múltiplos blocos de dados com checksums individuais. Cada bloco tem um par (start, end, checksum_offset).

RSA Signature

Firmwares mais novos têm assinatura RSA. Não pode ser corrigida sem a chave privada.

Mode $06 - Testes On-board

Resultados de testes de monitoramento on-board (SAE J1979 Mode $06)

Mode $06 retorna resultados de testes internos da ECU. Os valores são capturados antes de um DTC ser gerado, permitindo diagnóstico preventivo. Cada teste tem um TID (Test ID), valor medido, e limites min/max definidos pela ECU.

A base de dados é armazenada em formato JSON e salva no localStorage do navegador. Você pode importar/exportar arquivos JSON para backup ou compartilhar dados.

4. Importação em Lote

Para importar múltiplas entradas de uma vez, use um arquivo JSON com a seguinte estrutura:

{
  "type": "math-tools-export",
  "version": "1.0",
  "data": {
    "pinouts": [ ... ],
    "dtcCodes": [ ... ],
    "ecuMaps": [ ... ]
  }
}

5. Dicas

  • O campo "custom": true é adicionado automaticamente para diferenciar suas entradas das originais.
  • IDs devem ser únicos. Use formato "marca-modelo-descricao".
  • Exporte regularmente para não perder dados (localStorage tem limite de ~5MB).
  • Você pode editar o JSON exportado em qualquer editor de texto e reimportar.
